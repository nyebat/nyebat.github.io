Variabel

Variable Imutable

Variabel pada pemrograman rust dideklarasikan menggunakan kata kunci let.

Contoh

fn main() {
let a = 10;
let b = 3;
println!("{}", a + b); //output: 13 }

Variable mutable

Secara default, variabel pada rust bersifat imutable, untuk membuatnya menjadi mutable kita perlu menambahkan kata kunci let mut.

Contoh

fn main() {
let mut a = 10;
a = 17; //memperbarui nilai a menjadi 17
println!("{}", a); //output: 17 }
